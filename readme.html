トップページ

各種機能よりすべての機能の一覧が出ます。

カレンダーの日付をクリックすると、その日の予定の一覧が出てきます。

下の切り替えメニューより、年・月を切り替えることができます。

「今日以降のスケジュール一覧」より、受け取り指定しているカレンダーのスケジュール一覧を表示します。

「スケジュール追加」から、新しい予定を追加できます



日別スケジュール一覧

受け取り設定しているカレンダーから、選択した日付のスケジュールの一覧を表示します。

削除より、スケジュール削除ができます。



今日以降のスケジュール一覧

受け取り設定しているカレンダーから、今日以降のスケジュールの一覧を表示します。

削除より、スケジュール削除ができます。



スケジュール追加

受け取り設定しているカレンダーに、新しい予定を追加できます。

ただし、変更を許可していないカレンダーには追加できません。



受け取りカレンダー追加・削除

全てのカレンダーの一覧から、受け取るカレンダーの設定ができます。

ただし、個人用カレンダーは追加・削除できません。



ユーザー追加(Master・教職員・ユーザ登録IDのみ)

新規のユーザを追加できます。

所属チームで「教職員」を選択できるのは、Master・教職員ユーザのみです。

ユーザー登録用IDでは選択できません。



チーム追加(Master・教職員のみ)

新規のチームを追加できます。

生徒系列のみのチームが作成できます。



新規カレンダー追加(Master・教職員のみ)

新規のカレンダーを追加できます。

ただし、個人用カレンダーを追加することはできません。
